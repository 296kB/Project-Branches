<!-- GETTING STARTED -->
### Getting Started

## Clone the repo
   ```sh
   git clone https://github.com/296kB/Project-Branches
   ```

## Install all the dependencies or node packages used for development via Terminal

```sh
npm install 
```
---

## Start server

```sh
npm run dev
```
